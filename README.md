# bird-dog
Ai powered job hunting companion

## Overview
Bird Dog solves the problem of similar job titles hiding vastly different roles by analyzing JD's against your background, and helps you build a stronger application to roles that are actually worth your time. 

## Documentation
- [Product Requirements Document](Bird%20Dog%20PRD.pdf) - Problem definition and solution approach
- [System Specification](Bird%20Dog%20System%20Spec.pdf) - Technical architecture and implementation details

## Tech Stack
ChatGPT (initial ideation and execution of idea) Figma (Initial visual design) Claude (Built HTML code)

## What I Built
- Resume parsing and skills extraction
- JD analysis engine that identifies required vs. preferred qualifications
- Fit scoring algorithm comparing candidate background to role requirements
- Application strategy recommendations based on gap analysis
